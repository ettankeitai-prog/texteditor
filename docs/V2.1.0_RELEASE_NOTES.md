# Text Editor v2.1.0

## Added

- Codex App Serverをローカルstdioで利用するAI文章整形
- 「軽く整える」「正文化」「レビュー整形」の3種類のプリセット
- 元文と生成結果を確認する比較ダイアログ
- 選択範囲またはタブ全文への置換、新規タブ、クリップボードへのコピー
- ChatGPTアカウント認証、利用可能モデル、Codex利用枠の確認
- 明示パス、`where.exe`、Electron mainの`PATH`、Windows標準インストール先によるCodex CLI自動検出
- 本文・生成結果・認証情報を保存しないAI接続診断ログ

## Fixed

- 新規タブ作成後の非同期保存がEditor focusをタイトル欄へ移す問題
- AI Rewrite E2Eの不安定なpointer操作
- Codex App Serverへ渡すsandbox値を実プロトコルの`read-only`へ修正

## Notes

- ChatGPTプランに含まれるCodex利用枠を使用します。OpenAI APIキーによる従量課金には対応しません。
- AI出力は自動適用せず、比較ダイアログで確認してから適用します。
- Codex CLI未導入、未ログイン、利用枠到達、App Server異常時も通常の編集・保存・復元には影響しません。
